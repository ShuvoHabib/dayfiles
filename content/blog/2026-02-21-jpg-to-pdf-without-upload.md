## What to verify before you send the file

The final review should be short, but it should be disciplined. A good operator does not reread the whole file unless the task demands it. They check the few items most likely to break the workflow outcome.

1. Page order matches the intended sequence.
2. Images are readable after conversion.
3. The final PDF is small enough for the destination portal.

If those checks pass, the file is usually ready for the next handoff. If one fails, the problem is still caught early enough to fix without resetting the whole workflow.

## Common mistakes with JPG to PDF

1. **Dropping images in without sorting them first.** Rename or stage the images in order before conversion.
2. **Ignoring orientation differences between photos.** Rotate or crop awkward images before final PDF assembly.
3. **Assuming raw mobile photos are submission-ready.** Check margins, lighting, and readability before you create the PDF.
